# Laravel 5.6 Demo App

A Laravel App demonstration with Image Gallery

## Getting Started

### Installing

```
git clone https://github.com/NuBOXDevCom/laravel56-demo-app.git
```

And

```
composer install && npm install
```

## Running the tests

```
phpunit
```
